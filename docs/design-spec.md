### 布局说明

- **左侧 50%**：内容滚动区，包含五个公园的章节内容
- **右侧 50%**：地图固定（position: sticky），始终可见
- **导航条**：顶部 sticky，显示五个公园圆点指示器
- **交互逻辑**：左侧滚动到某个章节 → 右侧地图自动 flyTo 对应公园
- **响应式**：移动端改为上下布局（地图在上，内容在下）

## 技术栈

| 层 | 选型 | 理由 |
|----|------|------|
| HTML/CSS | 原生 | 无需框架，降低复杂度 |
| 地图 | Leaflet 1.9+ | 轻量、中文友好、文档全 |
| 底图 | OpenStreetMap + 天地图卫星图 | 免费、无需API Key |
| 图表 | ECharts 5 | 对比柱状图 |
| 语音 | Web Speech API | 浏览器内置，无需额外依赖 |
| 数据 | GeoJSON + Markdown/JSON | 公园边界 + 内容配置 |
| 部署 | Gitee Pages | 免费静态托管 |

## 核心功能模块实现方案

### 模块1：底图切换

- 默认加载 OSM 底图
- 右上角图层切换按钮：OSM / 天地图卫星
- 左下角显示：比例尺（Leaflet 内置）、缩放级别、鼠标经纬度
- 实现：Leaflet `L.control.scale()` + `mousemove` 事件

### 模块2：空间索引（公园定位）

- 加载 `parks.geojson`，5个公园用5种颜色区分
- 半透明填充（opacity: 0.3）+ 彩色边框（weight: 2）
- 三种选择方式：
  1. 点击地图上的公园多边形 → `polygon.on('click')`
  2. 侧边栏图例点击 → 触发 `flyTo(park.coords, park.zoom)`
  3. 空间分组按钮（西部/中部/东部）→ `fitBounds(groupBounds)`
- 选中效果：荧光色高亮（#00ff88, opacity: 0.6），其他公园恢复默认
- 右上角弹出公园名称标签（3秒自动消失）

### 模块3：详情展示

- 选中公园后，地图上方弹出折叠面板
- 6个子模块，点击展开独立内容区：
  1. **历史沿革**：时间线（纯CSS实现）
  2. **地理环境与生态特征**：二级折叠（地形/气候/水文/生态系统）
  3. **旗舰物种**：卡片网格，每物种图片+拉丁名+保护级别+习性
  4. **保护成效**：文字+关键数据高亮+数据来源
  5. **图片画廊**：CSS Grid，点击放大（lightbox）
  6. **相关文献**：外部链接列表，新标签页打开

### 模块4：分析对比

- 页面底部或专属区域"对比图表"按钮
- 点击弹出模态框，内含：
  - 面积对比柱状图（ECharts，降序排列，hover显示数值）
  - 旗舰物种数量对比柱状图
- 模态框底部标注数据来源

### 模块5：故事导览

- **手动导航**：页面固定底部的"上一站/下一站"按钮
  - 点击 → 地图 flyTo + 内容区 scrollTo 对应章节
  - 过渡文本弹出（5秒自动消失）
- **自动漫游**："开始漫游"按钮
  - 每5秒自动切换到下一个公园
  - 同步语音讲解（Web Speech API，浏览器TTS）
  - 按钮变为"暂停漫游"
- 循环：到武夷山后提示"漫游结束"

```json
  "type": "FeatureCollection",
  "features": [

      "type": "Feature",

        "color": "#4cc9f0",

        "zoom": 7
      },
      "geometry": { "type": "Polygon", "coordinates": [...] }
    }
  ]
}
```

### parks-content.json（由组员赖雅君等负责）

每个公园的文本内容，包括：
- 基础信息（名称、成立时间、位置、面积）
- 地理环境（地形、气候、水文、生态系统）
- 旗舰物种列表（名称、拉丁名、保护级别、习性描述、图片路径）
- 保护成效文本
- 参考文献链接列表

### 图片素材

- 每个公园至少5张高清图（公园景观 + 旗舰物种）
- 建议尺寸：1200×800 或 16:9
- 格式：JPG/WebP

## 交互逻辑

```
用户滚动页面
  ↓
IntersectionObserver 检测当前可见章节
  ↓
触发地图 flyTo(当前公园坐标)
  ↓
高亮当前公园边界
  ↓
更新导航条圆点状态
  ↓
（可选）点击"开始漫游"→ 自动按5秒间隔执行上述流程
```
